## What I Worked On

Before my frontend and developer-platform work, I worked on embedded and resource-constrained runtime systems at Sun Microsystems and Oracle.

## Technical Areas

- Java Embedded Suite
- RESTful services for embedded and distributed IoT environments
- Java Card VM
- Secure embedded JVM technologies
- Runtime architecture
- Feature scoping
- Performance optimization in constrained environments
- Platform portability and synchronization

## Why It Still Matters

This experience shaped how I think about execution systems, constraints, performance, determinism, and platform architecture. That mental model continues to influence my work on AI orchestration and developer tooling.
