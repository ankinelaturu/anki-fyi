# ZeroFabric Runtime Simulator

Lab experiment: an interactive simulator for **ZeroFabric** — exploring how intent maps to governed capability paths with policies, constraints, and observable execution.

## Goals

- Visualize active intent paths
- Compare deterministic vs adaptive routing
- Debug capability matching in real time

## Status

Prototype / work in progress. See **writing/zerofabric.md** for the conceptual model.
