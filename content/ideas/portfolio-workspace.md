# portfolio-workspace.md

## The Idea

This site is not intended to be a traditional portfolio. It is a professional knowledge workspace: markdown files, an IDE-like explorer, a terminal-style interface, and eventually a local AI assistant that answers questions from the same content used by the UI.

## Why Markdown

Markdown gives the site a clean content source. Add a new `.md` file and the workspace can automatically update the explorer, search index, and assistant corpus.

## Future AI Assistant

The next phase is local-first retrieval over this content, using browser-side embeddings and a local model such as Gemma/WebLLM where practical.
