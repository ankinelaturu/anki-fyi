# Stop Building Static Agents

Most "agent" demos are frozen graphs with a chat wrapper. Production systems need governed capability spaces, inspectable paths, and deterministic boundaries.

## Core argument

- Intent routing is not enough without constraints
- Capabilities should be explicit, versioned, and observable
- Humans need override and audit surfaces

## Related work

See **ZeroFabric** for a runtime coordination model and **Lintern** for local-first guardrails.
