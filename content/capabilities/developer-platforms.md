# developer-platforms.md

## Developer Platform Experience

Much of my career has focused on developer platforms and tools that make complex systems easier to build, inspect, and maintain.

## Relevant Systems

- Oracle Visual Builder
- Action Chain Editor
- Oracle Mobile Application Framework
- TypeScript enablement across runtime/build/tooling layers
- AI-assisted workflow generation

## Design Principle

A strong developer platform should help users move faster without hiding the execution model completely. The best tools are approachable, but still inspectable and debuggable.
