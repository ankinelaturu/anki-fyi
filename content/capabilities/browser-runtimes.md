# browser-runtimes.md

## Browser Runtime Interests

I am interested in the browser as an increasingly capable AI runtime: local inference, semantic matching, privacy-preserving inspection, DOM-aware tools, and interactive applications that run close to the user.

## Relevant Work

- Lintern Chrome extension and local-first detection model
- Ask Rudy / Rudy Finds browser-oriented retrieval experiments
- ONNX and local AI experimentation
- TypeScript-heavy interactive systems

## Why It Matters

Not every AI workflow should require sending user data to a remote service. Browser-native AI creates opportunities for privacy, low-latency feedback, and context-aware interaction.
