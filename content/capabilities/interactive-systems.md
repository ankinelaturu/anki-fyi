# interactive-systems.md

## Interactive Systems

I care about systems where UI is not just presentation. In my work, the interface often *is* the system: a visual programming canvas, an authoring environment, a retrieval surface, or a low-latency AI collaboration loop.

## Relevant Work

- Action Chain Editor
- Storybook Arcade
- OzMap learning trails
- Crafticle writing environment
- Lintern browser feedback loops

## Design Principle

The most valuable AI interfaces are not one-shot prompt boxes. They support iteration, inspection, correction, and continuous feedback.
