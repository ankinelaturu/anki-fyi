# rudy-finds.md

## What It Is

Rudy Finds is a semantic search style browser experiment for finding relevant content using AI-assisted retrieval.

## Technical Direction

- Semantic search over browser-accessible content
- Retrieval-focused UX
- Lightweight AI-assisted discovery
- Browser-native interaction patterns

## Why It Matters

Search should not always require exact keywords. Rudy Finds explores how semantic matching can make content discovery more natural and useful inside everyday workflows.
