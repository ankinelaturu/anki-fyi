# ozmap.md

## What It Is

OzMap is an AI-powered learning trail system that generates structured topic maps with branching exploration paths.

## Technical Highlights

- Graph-based learning orchestration
- Dynamic topic generation
- Branching exploration paths
- Synchronized execution state across sessions
- Complex client-side state management
- Real-time interactive learning navigation

## Why It Matters

OzMap explores how AI can become a guide through knowledge, not just a generator of isolated answers. It is about interactive exploration, user-driven paths, and structured discovery.
