# AstroValley

A browser-native AI sandbox disguised as an astrology application.

## What it explores

- Local LLM orchestration
- Browser runtimes
- Procedural 3D worlds
- React Three Fiber
- WebGPU/WebLLM experimentation
- Deterministic data-driven experiences

## Why it matters

AstroValley explores how rich AI-driven experiences can run close to the user, using the browser as the runtime.
