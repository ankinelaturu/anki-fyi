# crafticle.md

## What It Is

Crafticle is an AI-powered idea-to-article composer built around iterative human-AI collaboration.

## Technical Highlights

- React-based writing interface
- TipTap editor integration
- Structured idea extraction
- Content transformation pipelines
- Low-latency AI-assisted editing
- Human-guided article composition

## Why It Matters

Most writing tools treat AI as a one-shot generator. Crafticle explores a more iterative model where the user refines structure, intent, and output through a responsive editing experience.
